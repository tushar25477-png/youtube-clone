# YouTube Homepage Clone

I built this project to test my HTML and CSS skills by replicating the actual YouTube homepage layout from scratch. 

## The Cool Stuff
* **Fixed Header & Sidebar:** The top nav bar and side menu stay locked in place while the video section scrolls, just like the real site.
* **The Video Grid:** Used CSS Grid so the video cards automatically shift from 4 columns on a laptop down to 1 column on a phone screen.
* **Clean Hover Effects:** Added smooth transitions on the icons and video thumbnails when you mouse over them.

## The Hardest Parts I Solved
Getting the layout alignment right between the sidebar and the main video container without things overlapping took a lot of trial and error with CSS positioning. Reverting to custom spacing properties finally made it sit perfectly.
