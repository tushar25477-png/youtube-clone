# YouTube Clone
This is a frontend replica of the YouTube homepage layout built from scratch to practice responsive web design.

🛠️ Technologies Used
   HTML5 for semantic structure

   CSS3 for custom styling, Flexbox, and Grid layouts

✨ Features
   Fully responsive navigation bar and sidebar navigation

   Hover effects on video thumbnails (just like real YouTube)

   Grid layout that adjusts nicely from mobile screens to large desktop monitors
